* API para Products:


[GET]/api/products - Obtener todos los productos.
[GET]/api/products/:pid - Obtener producto segun su ID.
[POST]/api/products - Insertar nueva producto. Templates:
[PUT]/api/products/:pid - Actualizar un producto existente.
[DELETE]/api/products/:pid - Eliminar un producto existente.


* API para Carts:


[POST]/api/carts - Insertar nuevo carrito de compra.
[GET]/api/carts/:cid - Obtener carrito de compra segun su ID.
[PUT]/api/carts/:cid/product/:pid - Actualizar carrito de compra con nueva producto/item.
[DELETE]/api/carts/:cid/product/:pid - Eliminar un producto almacenado dentro de un carrito de compra.
